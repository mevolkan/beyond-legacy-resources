Resource Post Type plugin adds a Resource Post Type to your WordPress website.

== Description ==
The Resource Post Type plugin extends your WordPress website by adding a custom post type named 'Resource'. It allows you to create and manage resources easily.

== Features ==
- **Custom Post Type 'Resource':** Easily create and manage resources.
- **Seamless Integration:** Works with your existing WordPress theme.
- **Flexible Display Options:** Customize how resources are displayed on your site.

== Installation ==
1. Download the plugin files directly from [GitHub](https://github.com/mevolkan/beyond-legacy-resources).
2. Navigate to the WordPress admin dashboard.
3. Go to 'Plugins' > 'Add New' > 'Upload Plugin'.
4. Choose the downloaded plugin file and click 'Install Now'.
5. Once the installation is complete, click on 'Activate Plugin'.

== Usage ==
1. After activating the plugin, you will see a new menu item labeled 'Resources' in your WordPress dashboard.
2. Click on 'Add New' under 'Resources' to create a new resource post.
3. Add the title, content, and any additional information for your resource.
4. Publish the resource like any standard WordPress post.
5. Use shortcode [resource-loop] to display resources in your content.
6. Use shortcode [combined-loop] to display both the filter for resources and filter.

== Customization ==
- **Template Overrides:** Override the default templates by creating template files in your theme directory.
- **Styling Resources:** Add custom CSS in your theme to style the resource posts.

== Requirements ==
- WordPress 6.2
- PHP 8.1 or higher.

== Notes ==
- Ensure your WordPress theme is compatible with custom post types.
- Check for any plugin updates regularly for improved security and functionality.
